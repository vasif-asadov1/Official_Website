## 🛠️ Local Development

If you wish to clone this repository and run the website locally:

1. **Clone the repository:**
   ```bash
   git clone git@github.com:vasif-asadov1/Official_Website.git
   cd Official_Website
   ```
2. **Install MkDocs** (assuming Python is installed):
   ```bash
   pip install mkdocs-material
   ```
3. **Serve the site locally:**
   ```bash
   mkdocs serve
   ```

The site will be available at http://127.0.0.1:8000/


## 🚀 Deployment

This website is automatically deployed to GitHub Pages. Any new documentation added to the `docs/` folder can be published using:
```bash
mkdocs gh-deploy
```
